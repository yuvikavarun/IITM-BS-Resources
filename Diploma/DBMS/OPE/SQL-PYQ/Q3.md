Write an SQL query to find the match dates where the host team score is not between 1 and 4

<img width="831" height="731" alt="flis" src="https://github.com/user-attachments/assets/462d3e3c-a007-4a74-b03c-87c6ca5c8203" />

Solution:
```
SELECT match_date
FROM matches
WHERE host_team_score NOT BETWEEN 1 AND 4
```
